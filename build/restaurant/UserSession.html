Renvoi le contenu du tableau de Session dans une variable Render. Cette variable doit être retournée dans la méthode HttpGetMethod du Controller pour permettre le maintien de la connexion. La commande a exécuter dans le controller est la suivante : $render = (new UserSession)->getAll(); Cette commande doit être présente au départ de chacune des class de type controller.
